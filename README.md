## âœï¸ Customization

### Adding Your Work
1. Copy any portfolio item section in `index.html`
2. Update the icon, title, and description
3. Add your project links and file downloads

### Updating Content
- Replace placeholder text in the About section
- Update contact information
- Add your social media links
- Upload your images to the `assets/images/` folder

### Changing Colors
The website uses Tailwind CSS classes. Main colors:
- Primary pink: `text-pink-300`, `bg-pink-300`
- Hover states: `hover:text-pink-400`
- Backgrounds: `bg-pink-100`

### Stopping the Server
Press `Ctrl+C` in the terminal to stop the server.

### Troubleshooting
- **Port already in use**: The server will show an error if port 8000 is busy
- **Python not found**: Install Python 3 from python.org
- **Permission denied**: Run `chmod +x start.sh` to make the script executable
